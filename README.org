* spacemacs
** 使用
#+BEGIN_SRC shell
# 克隆 spacemacs 的配置包
git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d
# 克隆当前包
git clone git@github.com:haormj/spacemacs.d.git ~/.spacemacs.d
#+END_SRC
** 支持中文输入
直接将系统的 emacs 软链接换为如下脚本
#+BEGIN_SRC shell
# !/bin/bash
LC_CTYPE=zh_CN.UTF-8 emacs-26.1
#+END_SRC
